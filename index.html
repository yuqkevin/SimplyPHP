Welcome to SimplyPHP!

SimplyPHP is a simple, fast and scalable MVC PHP framework. Simplicity does not necessarily mean poor in power. On the contrary, Simplicity means straightforward, short learning curve for newbie, more flexibility for experienced developer.

Features

  • Scalable multi-tier architecture
  • MVC design pattern
  • Component-oriented UI: Every request (URL) maps to a particular component
  • Business-oriented structure: All components are organized in business structures - W3S Bean
  • OO style database CRUD operations: SQL is no longer mandatory for data access
  • Easy configuration: PHP-style configuration file
  • Simple and Straightforward: learning to write database-driven application in 15 minutes!

Key Concepts

  • Component

    It is a function unit in UI. Page is a special component: container. Every component is mapped to a particular pair of <Model, Handler>. From point view of programmer, the Component is also an included handler within Model. So sometimes we also call handler as Method. The View is wrapper of Component.

  • Bean

    It is a structure contains files for particular application: Component set <model, handler, view>, Library class(business logic) and Database(table) definition.

  • Table Object

    In SimplyPHP fromework, all database tables and views are objects under table hook, record is associative array which table object deals with. All CURD operations are implemented as table object's methods and no SQL needed at all.
