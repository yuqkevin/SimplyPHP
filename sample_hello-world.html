Home:: SimplyPHP:: Learn by example I: Hello World!

Requirements

  • A page with content "Hello World!"

Database

No database needed in this case.

Bean & Component

  • Bean
    • Root Bean: For our convenience, we create a bean named sample to contain all samples. 
      cd document_root/system/app/resource/beans
mkdir sample
cd sample
vi model.class.php
# lib class can be omitted in this case which has no db access
vi lib.class.php
      model.class.php 
      <?php
class Sample extends Model
{
}
      lib.class.php (optional in this case which has no db access) 
      <?php
class LibSample extends Library
{
}
      At this moment we haven't defined any method in classes above.
    • Child Bean: Then we create a child bean hello under the bean sample: 
      # create sub-bean hello
mkdir hello
cd hello
# create model and lib class
vi model.class.php
# lib class can be omitted in this case which has no db access
vi lib.class.php
      model.class.php 
      <?php
class SampleHello extends Sample
{
}
      lib.class.php (optional in this case which has no db access) 
      <?php
class LibSample extends LibSample
{
}
  • Component

    Now we are going to define the component for request of "hello world":
    <Model, Handler, View> = <SampleHello, world, view/world.tpl.php> 

    # create handler folder
mkdir handler
# handler file: 
# world.inc.php can be omitted in this case which has no logic process.
touch handler/world.inc.php
# create view folder
mkdir view
# edit view
vi view/world.tpl.php
    view/world.tpl.php 
    Hello World!

  • URL: http://your_simplyphp.domain.com/SampleHello/world

    Every request in SimplyPHP is always a request on particular component.

  • File Structure overview of bean sample and child bean hello: 
    beans/--+--sample/--+--model.class.php,lib.class.php
                    |
                    +--hello/--+--model.class.php,lib.class.php
                               |
                               +--handler/
                               |
                               +--view/--+--world.tpl.php

Access Control: Bean & Model

  • Enable the Bean: copy or symbol link root bean sample from app/resource/beans folder to app/beans folder 
    cd document_root/system/app/beans
ln -s ../resource/beans/sample .
  • Enable Model: We can add model SampleHello to model access array in domain configure file your-simplyPHP.domain.com.ini 
    [Access]
model[] = SampleHello
; or use wildcard
model[] = Sample*

Finished

That's it! Now you try http://your_simplyphp.domain.com/SampleHello/world and will get content with "Hello World!"
