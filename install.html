Installation

# 1. download & extract
$ wget -O simplyphp.tar.gz https://github.com/yuqkevin/SimplyPHP/tarball/master
$ tar -xvf yuqkevin-SimplyPHP-nnnn.tar.gz

# 2. mv files into web document root folder
$ mkdir -p /var/sites
$ mv yuqkevin-SimplyPHP-nnnn /var/sites/w3s

# 3. virtual host configuration in apache configure file:

    ServerName      your_w3s_site.domain.com
    documentRoot    /var/sites/w3s/webroot


# 4. Reload apache
$ sudo /etc/init.d/apache2 reload

# 5. In you browser, input address: http://your_w3s_site.domain.com
#    The welcome page will be shown up. 
#    Congratulations! You've installed SimplyPHP successfully!

Application Configuration

For the quick installation, you don't need touch configure file at this moment. But an overview of configuration file will be helpful for the practice we are going to do.

Files: webroot/system/app/conf/main.ini(default for all domains), user.domain.com.ini (for particular domain):

[Global]
; Global configurable parameters
...
[Access]
; Directly accessable models via http(s) (multiple lines if needed):
; model[] = ModelName
...
[Domain]
; Model access resitriction on domain
; following line set ModelNameA only accessable for domain1.com and domain2.com
; ModelNameA = "domain1.com,domain2.com"
...
[Model]
; Default URL mapping for model
; following lines will set model welcome as default model for all request
; but for request like /user/..., the UserModel will be override default Welcome model.
; Welcome = "/"
; UserModel = "/user"
...
[Database]
; database connection parameters.
...
