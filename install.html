Installation

# 1. download & extract
$ wget -O simplyphp.tar.gz https://github.com/yuqkevin/SimplyPHP/tarball/master
$ tar -xvf yuqkevin-SimplyPHP-nnnn.tar.gz

# 2. mv files into web document root folder
$ mkdir -p /var/sites
$ mv yuqkevin-SimplyPHP-nnnn /var/sites/w3s

# 3. virtual host configure in apache configure file:

	ServerName		your_w3s_site.domain.com
	documentRoot	/var/sites/w3s/webroot


# 4. Reload apache
$ sudo /etc/init.d/apache2 reload

# 5. In you browser, input address: http://your_w3s_site.domain.com
#    An welcome page will be shown up. 
#    Congradulations! You've install SimplyPHP successfully!
